# GourtmetGO

En este repositorio de tiene la interfaz del usuario dueño de restaurante. 
## Tech Stack

**Client:** Next.js, HTML, TailwindCSS, CSS

## Features

- Pantalla principal
- Selección de plan
- Registro de usuario
- Login de usuario
- Pantalla de Pedidos
- Pantalla de Menú
- Pantalla de Gestión de Empleados
- Pantalla de Método de Pago
